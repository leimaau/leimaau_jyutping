# myself_jyutping

 **jyutping**即拼方案――是原来rime输入法自带的方案，没有声调。

 **hungmaau_jyutping**即熊猫粤拼――带声调，我制作的方案，收字音范围为统一码组织统汉字库的所有读音，兼收粤语审音配词字库的读音以及香港增补字符集的读音，只有繁简体转换。

 **leimaau_jyutping**即狸猫粤拼――带声调，我制作的的方案，收字音范围同上，提供传统繁体，大陆简体，台湾繁体，香港繁体，日本繁体转换。

两份方案的词典部分收录了《开放粤语词典》以及CC-Canto提供的词汇。

**注意：**

同文输入法中opencc的位置在 rime\opencc

小狼毫输入法中opencc的位置在 程序文件夹\data\opencc
