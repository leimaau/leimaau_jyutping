# myself_jyutping

拼方案--是原硗文入法搭配的方案，不{

V州方案--{，我的方案，收字^窄，有些生僻字在手C@示不正常

V府方案--{，我的方案，收字^V，有些生僻字在手C@示不正常

注意：方案用在同文入法中要加入opencc，如果要用到小狼毫要自己把opencc的jsonDQ改成ini文件

例如：

```
zh_simp:
  option_name: zh_simp
  opencc_config: t2s.json
  tips: all
```

  
改成

```
zh_simp:
  option_name: zh_simp
  opencc_config: t2s.ini
  tips: all
```
