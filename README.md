# myself_jyutping

 **jyutping**即拼方案――是原来rime输入法自带的方案，没有声调。

 **myself_jyutping_strict**即V州方案――带声调，我的方案，收字范围较窄，收录字音较严格，生僻字在手机显示不正常的情况较少，但多音字注音也未必正确，只能作参考，所以万万不能当作学习正音的工具。

收字音范围为《粤语审音配词字库》的非异读音，以及《1992年常用字广州话读音表》的读音。

 **myself_jyutping**即V府方案（多字w）――带声调，我的方案，收字范围较广，收录字音较多，会有生僻字在手机显示不正常的情况，多音字注音未必正确，也不能当学习正音的工具。
 
  **myself_jyutping2**即V府方案（繁w）――同上，不过只有繁简体转换，不处理多种字体，在PC端的文件中，移动端没有，需要的话可以自己改写。

收字音范围为《粤库》的所有读音以及《1992年常用字广州话读音表》的读音，还收录了《粤语同音字典》，《广州音字汇》，包括了不再上述资料而在原粤拼方案的字音，但是不带声调，另外还收录了香港增补字符集的字音。

两份方案的词典部分收录了《开放粤语词典》提供的词汇。

两份方案补充了缺失的传统字体，只有台湾字体的部分字，例如「」字。

**注意：**

同文输入法中opencc的位置在 rime\opencc

小狼毫输入法中opencc的位置在 程序文件夹\data\opencc
